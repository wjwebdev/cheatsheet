HTML5 Notes

HTML (HyperText Markup Language) provides the structure for content on a web page and is the most basic building block of the web. Hypertext refers to links that connect web pages to one another. Either within a single website or between websites. HTML tags have been listed in a table below.

HTML Tags Information
<p> Defines a paragraph
<head> Defines the head of the document, this is used to hold the metadata information for a webpage.
<title> Title tag defines the title for a webpage and can be seen on the browsers tab.
<body> The body tag represents the content for a webpage and only one body tag is required.
<header> is a semantic tag that normally represents introductory information on a webpage.
<footer> footer element contains information about the author or contact information. And will normally be added at the bottom of a webpage.
<article> Article tag represents a self contained composition in a document.
<section> A semantic element that represents a generic standalone section of a document.
<div> A generic container for content
<span> The span HTML element is a generic inline container for phrasing content
<img> Embeds an image onto a webpage and must include a src attribute.

CSS3 Notes

CSS (Cascading Style Sheet) is used to provide style to a web page. Without CSS, a webpage looks very bland and empty with just text and images on a white background. With the power of CSS we can build websites that are visually appealing. CSS3 properties and values are listed in a table below.

CSS Properties Information
Opacity: Opacity property is used to set the opacity of a selected element, allowing you to control if content behind that element is visible.
margin: Sets the margin area on all four sides of an element at once.
Padding: Sets the padding area on all four sides of an element at once.
text-align: Sets the horizontal alignment of text inside a block level element.
font-size: Sets the property size of the font
font-weight: Sets the font weight for a font-family
text-transform: Specifies how to capitalise text
border: Shorthand property that sets the elements border
border-radius: Border-radius property rounds the corners of an elements outer border edge.
line-height: Sets the height of a line. Commonly used for paragraph text on a webpage.
box-sizing: Sets how the total width and height of an element is calculated.
